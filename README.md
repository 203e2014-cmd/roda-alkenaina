# موقع روضة الكنانية الخاص (صفحة واحدة)

هذا مشروع موقع بسيط (HTML/CSS/JS) جاهز للنشر كموقع تعريفي لروضة الكنانية الخاص.

## تشغيل سريع (على الكمبيوتر)
- افتح ملف `index.html` في المتصفح مباشرة.
- أو شغّل أي سيرفر محلي بسيط (اختياري).

## تعديل المحتوى
كل النصوص داخل `index.html`. يمكنك تعديل:
- **ساعات العمل**
- **العنوان**
- **البرامج والأنشطة**
- **أي نصوص أخرى**

## إضافة صور للمعرض
الموقع الآن يحتوي على **صور تجريبية** داخل `assets/gallery/`.

إذا أردت إضافة صور:
1) أنشئ مجلد مثل: `assets/`
2) ضع الصور داخله (مثال: `assets/1.jpg`)
3) في قسم **معرض الصور** داخل `index.html` استبدل روابط الصور الحالية بروابط صورك، مثل:

```html
<img class="gallery__item" src="./assets/1.jpg" alt="نشاط داخل الروضة" />
```

> يمكنك تعديل ارتفاع الصور من ملف `styles.css` عبر الخاصية `height` داخل `.gallery__item`.

## إضافة خريطة Google
داخل قسم **العنوان والخريطة** ستجد تعليقًا يوضح أين تضع رابط الـ Embed.

الخطوات:
1) افتح Google Maps
2) ابحث عن موقع الروضة
3) مشاركة (Share) → تضمين خريطة (Embed a map) → انسخ الرابط داخل `src` للـ iframe
